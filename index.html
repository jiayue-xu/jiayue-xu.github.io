string

发表于 2023-04-16 分类于 cpp

IO操作

  • cin >> s - 从标准输入读入字符串至s中，在遇到第一个有效字符前，会自动忽略空格和回车，遇到第一个有效字符后开始读入字符串，直到遇见第一个空格或回车停止读取，并将空格和回车留在缓冲区内。

  • getline - 从第一个字符开始读取，直到遇见回车停止读取，并把回车丢弃。

  • s.empty() - 返回true如果字符串为空

  • s.size() - 返回字符个数

    • 不要用这个函数来判断字符串是否为空，会影响代码可读性
    • 用string::size_type类型的变量来接收size()的返回值，这是一个独立于平台的类型，实际上是一个unsigned类型
    • 不要把返回值和signed类型的变量进行比较，容易引起意想不到的结果

cctype

在C标准库<ctype.h>中定义了很多函数简化字符串的操作，在C++中可以通过包含<cctype>头文件使用这些函数。使用<cctype>和<ctype.h>的区别在于，把这些函数添加到了std::命名空间中，有利于写法的统一，避免命名冲突。

<cctype>中常用的函数有：

  • isalnum(c) - 如果c为字母或数字，返回true
  • isalpha(c) - 如果c是字母，返回true
  • isdigit(c) - 如果c是数字，返回true
  • islower(c) - 如果c是小写字母，返回true
  • ispunct(c) - 如果c是标点，返回true
  • isspace(c) - 如果c是空格、回车、tab，返回true
  • isupper(c) - 如果c是大写字母，返回true
  • isxdigit(c) - 如果c是十六进制数字，返回true
  • tolower(c) - 如果c是大写字母，把它转换为小写字母
  • toupper(c) - 如果c是小写字母，把它转换为大写字母
